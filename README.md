# Algorithm Visualizer & Data Manager Web App

A modular, full-stack web application designed to manage multi-datatype records and visualize core computer science algorithms. Built with an offline-first architecture, this system operates completely locally without requiring an internet connection or external server configurations.

## 🚀 Key Features
* **Multi-Datatype Storage:** Form interface capable of capturing and validating 5 distinct data types (Integer, String, Float, Date, and Boolean).
* **Custom Algorithm Integration:** Pure Python implementations of sorting and searching algorithms to manipulate database memory dynamically.
* **Responsive Control Panel:** User dashboard designed with variable input queries matching explicit course specifications.
* **Graceful Failure Handling:** Active UI handling for empty states, explicitly rendering a "no data" placeholder when zero match criteria are met.

---

## 🛠️ Tech Stack & Architecture
* **Language:** Python
* **Framework:** Flask (Micro-framework)
* **Database:** SQLite (Self-contained, zero-configuration relational storage)
* **Frontend:** HTML5 / CSS3 via Jinja2 template rendering
